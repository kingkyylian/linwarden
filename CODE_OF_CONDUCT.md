# Code of Conduct

Linwarden contributors are expected to keep discussions focused, respectful, and useful.

## Expected Behavior

- Critique code and ideas, not people.
- Keep issue and PR comments actionable.
- Assume operational context may vary across distributions.
- Disclose security-sensitive details privately.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private security reports before maintainers can respond.
- Repeated off-topic comments that block technical work.

Maintainers may edit, hide, or remove comments and contributions that violate these expectations.
